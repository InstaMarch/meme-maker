# meme-maker
Shell scripts for creating memes:
- raw2ppm
- make-caption
- make-composite

## pre-reqs
- 
dcraw
imagemagick

## examples
### convertes raw format to ppm intermediate file format
./raw2ppm.sh image.rw2 > image.ppm

### Resize image.png to standard size
./resize.sh

### sizes and converts the image to png
./make-text.sh

### Composite
./make-composite
